# README

## 開発言語
* Ruby version: 3.0.1
* Rails 6.0.3

## 実行手順
* $ git clone git@github.com:Roatgithub/Mimamori.git
* $ cd mimamori
* $ bundle install
* $ yarn install --check-files
* $ rails db:create
* $ rails db:migrate
* $ rails s
